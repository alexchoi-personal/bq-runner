⏺ Use the /export command:

  /export [filename]

  - /export — exports to clipboard or prompts for a location
  - /export my-conversation.md — exports to a specific file

  If you want to export a previous session, you can resume it first with /resume or claude -c, then run /export.

⏺ No built-in auto-export setting, but you can use a SessionEnd hook.

  Add this to .claude/settings.json:

  {
    "hooks": {
      "SessionEnd": [
        {
          "hooks": [
            {
              "type": "command",
              "command": "your-export-script.sh"
            }
          ]
        }
      ]
    }
  }

  The hook receives context including:
  - session_id
  - transcript_path — path to the conversation JSON
  - reason — why session ended

  Example script to copy transcripts:
  #!/bin/bash
  TRANSCRIPT_PATH=$(jq -r '.transcript_path' <<< "$1")
  cp "$TRANSCRIPT_PATH" ~/exported-conversations/

  If you want this as a native feature, you can submit a request with /feedback.
